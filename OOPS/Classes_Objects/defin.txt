self :-- 
In Python, self is a conventional name for the first parameter of a method within a class definition. 
It serves as a reference to the instance of the class on which the method is being called. 

In Python, self is like the glue that binds data to behavior inside a class.
It refers to the instance of the class itself,
allowing you to access and modify the object’s attributes and methods.
